# 🎓 AI Career Guidance Chatbot

An AI-powered Career Guidance Chatbot built using **Python**, **Streamlit**, and **Google Gemini API**. This application helps students and job seekers receive personalized career recommendations, skill roadmaps, project ideas, certification guidance, and interview preparation tips through an interactive chat interface.

## 🚀 Features

* 🤖 AI-powered career guidance using Gemini API
* 🎯 Personalized career recommendations
* 🛣️ Learning roadmap generation
* 💡 Project suggestions based on career goals
* 📜 Certification recommendations
* 🎤 Interview preparation tips
* 💬 Interactive chat interface
* 📝 Chat history tracking
* 🎨 Modern and user-friendly Streamlit UI

## 🛠️ Tech Stack

* Python
* Streamlit
* Google Gemini API
* Python Dotenv
* Datetime

## ⚙️ Installation

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/AI-Career-Guidance-Chatbot.git
cd AI-Career-Guidance-Chatbot
```

### 2. Install Dependencies

```bash
pip install -r requirements.txt
```

### 3. Create Environment File

Create a `.env` file in the project root directory.

```env
GEMINI_API_KEY=YOUR_API_KEY_HERE
```

### 4. Run the Application

```bash
streamlit run app.py
```
